## How to Use

1. **Open the Application**: Simply open `index.html` in your web browser
2. **Browse Events**: Scroll through the complete timetable to see all scheduled events
3. **Filter by Day**: Click on the day buttons at the top to view specific day schedules
4. **View Event Details**: Each event card shows:
   - Event time
   - Event title
   - Speaker name(s)
   - Event description
   - Location/venue
